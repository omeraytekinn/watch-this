# Installation

```
git clone https://github.com/omeraytekinn/watch-this.git
```

# Run

## Environment

```
cd watch-this
pip install pipenv
pipenv shell
pip install -r requirements.txt
```

## Run crawler

```
python app/src/crawler.py
```

## Run app

```
python run.py
```
